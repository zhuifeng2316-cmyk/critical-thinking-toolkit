# 批判性思维框架
## Critical Thinking Framework

> 不给答案，给思维框架

---

## 🧠 五个核心心智模型

### 1. 第一性原理思维

**定义**：回归事物本质，不依赖类比和经验

**公式**：
```
问题 → 拆解到基本事实 → 从基本事实重新构建解决方案
```

**使用场景**：
- 需要创新时
- 别人说"一直这样"时
- 成本过高时

**实例**：
```
传统思维：电动车电池贵，因为电动车都贵
第一性原理：电池 = 镍 + 钴 + 铝，原材料成本X，为什么卖Y？
结论：规模化生产可以降低成本 → 特斯拉
```

**触发词**：`「第一性原理」「本质是什么」「拆解」`

---

### 2. 二阶思维

**定义**：不只看结果，看结果的结果

**公式**：
```
行动 → 一阶后果 → 二阶后果 → 三阶后果
```

**使用场景**：
- 做重大决策时
- 看起来太好的事
- 有副作用的风险

**实例**：
```
一阶：降价 → 销量上涨
二阶：利润下降 → 品牌贬值
三阶：客户认为你廉价 → 难以提价

结论：短期好，长期坏
```

**触发词**：`「二阶思维」「然后呢」「后果的后果」`

---

### 3. 逆向思维

**定义**：与其追求成功，不如避免失败

**公式**：
```
目标 → 如何失败 → 避免失败的方式 → 成功
```

**使用场景**：
- 正向思考卡住时
- 风险评估时
- 制定计划时

**实例**：
```
目标：创业成功
逆向：如何让创业必死？
答案：不验证需求、不做差异化、不听用户反馈
正向：避免这些 → 成功率提高
```

**触发词**：`「逆向思维」「如何失败」「避免什么」`

---

### 4. 概率思维

**定义**：世界不是确定性的，是概率分布的

**公式**：
```
决策 = 期望值 = 概率 × 收益
```

**使用场景**：
- 不确定性决策
- 评估机会
- 风险管理

**实例**：
```
选项A：100%概率赚10万 → 期望值 = 10万
选项B：50%概率赚30万，50%概率亏5万 → 期望值 = 12.5万

理性选择：B（但要看你的风险承受能力）
```

**触发词**：`「概率思维」「期望值」「不确定性」`

---

### 5. 系统思维

**定义**：看整体，不看局部；看关系，不看元素

**公式**：
```
元素 → 关系 → 结构 → 行为 → 模式
```

**使用场景**：
- 问题反复出现时
- 改一个地方影响其他地方
- 需要"治本"时

**实例**：
```
问题：员工离职率高
元素视角：提高工资（治标）
系统视角：
  - 招聘标准是否匹配？
  - 培训是否到位？
  - 晋升通道是否清晰？
  - 文化是否认同？
结论：系统性解决
```

**触发词**：`「系统思维」「整体视角」「循环关系」`

---

## 📋 使用指南

### 快速检查清单

当面临决策时，问自己：

```
□ 我用的是第一性原理还是类比？
□ 我考虑了二阶后果吗？
□ 我逆向思考过吗？
□ 我评估了概率吗？
□ 我看了整体系统吗？
```

### 组合使用

```
复杂问题 → 第一性原理拆解
         → 逆向思维找坑
         → 二阶思维看后果
         → 概率思维评估
         → 系统思维整合
```
