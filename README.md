## 🚀 快速开始

### 1. 批判性思维五问

当面临决策时，问自己：

```
□ 我用的是第一性原理还是类比？
□ 我考虑了二阶后果吗？
□ 我逆向思考过吗？
□ 我评估了概率吗？
□ 我看了整体系统吗？
```

### 2. 决策矩阵

有多个选项时，用量化方式决策：

| 选项 | 收益(40%) | 成本(30%) | 风险(20%) | 时间(10%) | 总分 |

## 📖 为什么做这个？

很多人接受的教育是：记住标准答案。

但真实世界没有标准答案。

你需要的是：
- 拆解问题的能力
- 评估选项的能力
- 做出决策的能力

这个项目，就是帮你建立这些能力。
